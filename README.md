Blazor and ASP.NET Core Web API application that allows its users to manage collections of characters, teams, and powers.

Features:
  Add, view, and delete entries across all three data sets
  Validation to enforce system rules and constraints
  RESTful API Design
  Entity Framework Core implementation for database interaction

Technologies:
  .NET 6.0 or later
  Blazor
  ASP.NET Core Web API
  Entity Framework Core
  Microsoft SQL Server

Setup:
  Clone repo
  Ensure LocalDB is ready (connection string can be modified in appsettings.json)
  Run: dotnet ef database update
  Set TeenTitansRP.API and TeenTitansApp as startup projects
  Run

Future Implementations:
  Additional validation checks
  Stylistic enhancements
  Update functionality
