# Teen-Titans-Blazor-Project
Blazor and ASP.NET Core Web API application that manages characters, powers, and teams associated with the teen titans franchise.

Features:
  RESTful API
  Get/Add/Delete Characters
  Get/Add/Delete Teams
  Get/Add/Delete Powers
  Client and server validation

Technologies Required:
  .NET 6 or higher
  ASP.NET Core Web API
  Blazor
  Entity Framework Core
  SQL Server LocalDB

Set-Up:
  Clone repo
  Run: dotnet ef database update (refer to appsetting.json for modification of connection string)
  Enable multiple projects for startup (TeenTitansRP.API, TeenTitansApp)

Possible Future Updates:
  Implement relationships between all three entities
  Stylistic improvements
  Update functionality
  Additional validation responses
